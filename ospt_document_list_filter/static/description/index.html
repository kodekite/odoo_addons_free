Module Detail

Basicly odoo only show all document to every user

in example when you create invoice document every user can see your document, not only their own document but all of documents

This module used to filter document based on user created and user who assigned to that document and also based on setting user who allowed to look all data in tree

In example, Create an Invoice with user created is Administrator And SalesPerson is also Administator


List / Tree Invoice will be like picture above (basicly in every users)


If You already installed this module, Another user besides (1. user who created the document, 2. user who assign in document, 3. user allowed in setting) will not see the document created (In this case user named jacky)


If You want user allowed to see all document, just add that user to configuration in Configuration Sub menu


After you add that user to allowed user in configurations, the user will be able to see the document even though that user is not user created and user assigned to the document


This module only cover invoice document, you can do to every module self but if you can't, just contact me to customize :)
